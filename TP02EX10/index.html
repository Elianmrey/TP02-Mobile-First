Questão sobre Media Queries

Qual das seguintes opções descreve corretamente o que é uma media query em CSS?

  • 1. Uma função JavaScript que altera o layout da página.
  • 2. Um método para importar fontes externas para a página.
  • 3. Um mecanismo para aplicar estilos CSS baseados em características do dispositivo, como largura de tela.
  • 4. Uma técnica para animar elementos HTML.
  • 5. Um recurso para definir variáveis CSS que podem ser reutilizadas.

Resposta correta: 3. Um mecanismo para aplicar estilos CSS baseados em características do dispositivo, como largura de tela.

Justificativa: As media queries permitem que os desenvolvedores apliquem diferentes estilos CSS dependendo das características do dispositivo que está sendo usado, como largura de tela, resolução, ou orientação. Isso é fundamental para o design responsivo, onde o objetivo é garantir que o site funcione bem em diversos dispositivos e tamanhos de tela.
