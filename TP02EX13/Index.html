1. O que é Mobile-First?




Resposta Correta: C) Desenvolvimento focado primeiro em dispositivos móveis.

Justificativa: Mobile-First é uma abordagem onde o design é criado inicialmente para dispositivos móveis, garantindo uma boa experiência para os usuários.

2. O que é uma Media Query?




Resposta Correta: B) Um recurso CSS usado para aplicar estilos diferentes com base em características do dispositivo, como largura da tela.

Justificativa: Media Queries permitem a criação de layouts responsivos ao adaptar o design para diferentes tamanhos de tela.

3. Qual é a ordem ideal para o desenvolvimento utilizando a abordagem Mobile-First?




Resposta Correta: C) Criar o layout para dispositivos móveis e depois expandir para telas maiores.

Justificativa: A abordagem Mobile-First foca em projetar para dispositivos móveis primeiro, garantindo um design simples e eficiente, e depois expande o design para telas maiores.

4. Qual das opções abaixo representa um exemplo de Media Query que se aplica a telas com largura mínima de 768px?




Resposta Correta: B) `@media only screen and (min-width: 768px)`

Justificativa: A Media Query `min-width` define estilos que se aplicam quando a largura da tela é igual ou superior a 768px, tipicamente utilizada para tablets e dispositivos maiores.

5. O que significa o termo "Responsividade" em web design?




Resposta Correta: B) Capacidade de uma página web de ajustar seu layout e conteúdo de acordo com o dispositivo do usuário.

Justificativa: Responsividade refere-se à habilidade de um site de adaptar seu layout de forma eficiente para diferentes dispositivos, proporcionando uma experiência de usuário consistente.
