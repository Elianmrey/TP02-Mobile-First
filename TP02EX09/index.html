Teste de Conhecimento sobre Responsividade

Pergunta:

Qual das abordagens abaixo é a mais recomendada para começar a criar um design responsivo?

  1. Definir primeiro o layout para desktops e depois ajustar para dispositivos móveis.
  2. Usar imagens em alta resolução sem se preocupar com o tamanho do dispositivo.
  3. Começar o design pensando nos dispositivos móveis e, em seguida, adaptá-lo para telas maiores.
  4. Sempre utilizar tamanhos fixos para elementos no layout.
  5. Ignorar o uso de media queries e focar apenas no design de um único tipo de tela.

Resposta correta: Começar o design pensando nos dispositivos móveis e, em seguida, adaptá-lo para telas maiores.

Justificativa: A abordagem "Mobile-first" é amplamente recomendada, pois prioriza a experiência em dispositivos móveis, que representam a maior parte do tráfego na web atualmente. Depois de otimizar para o mobile, é mais fácil expandir e ajustar o design para telas maiores.
